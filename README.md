## Summary
<p> Desafio Warren - Web </p> 
O desafio será implementar uma tela de lista de transações apresentando os principais</br>
dados relacionado a cada uma delas, ao clicar em uma transação será exibido um modal </br>
contendo os detalhes desta transação.</br>
 
## Command

 Run => npm start </br>
 Build => npm run build </br>
 Test: => npm run test </br>
